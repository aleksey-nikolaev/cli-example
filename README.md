# Simple CLI 

Интерфейс командной строки для манипулирования изображениями с помощью [Image Magick](https://imagemagick.org).

Этот проект - тестовое задание 

#### Тестовое задание для кандидатов в команду разработки компании Аргус
Тестовое задание для проверки навыков: 
* проектирования
* программирования, 
* работы с библиотеками и чужим кодом

Требуется:
* Разработать приложение осуществляющее обработку изображения.
* Приложение консольное.
* После запуска в консоли доступны команды из списка (приведен в приложении)
* Загрузку, сохранение и обработку изображений реализовать средствами Image Magick

Ожидаемый результат:
* Результат выполнения в виде ссылки на репо в github
* Платформа, среда разработки - по выбору (windows/linux/macos). 
* Язык разработки - С++

##### Приложение
|Команды/Команда|параметры|действие
||||
|load, ld|`<name>` - имя изображения, по имени оно доступно в других командах, `<filename>` - имя файла для загрузки|Загрузка изображения в формате JPG|
|store, s|`<name>` - имя изображения, `<filename>` - имя файла для сохранения|Сохранение изображения в формате jpg|
|blur|`<from_name>` - имя изображения, `<to_name>` - имя изображения, `<size>` - размер сглаживаемой области|Сглаживание изображение|
|resize|`<from_name>` - имя изображения, `<to_name>` - имя изображения, `<new_width>`, `<new_height>`|Изменение размера изображения|
|help, h||Выводит справку о поддерживаемых командах|
|exit, quit, q||выход|

##### Пример сессии:
```
ld img1 image.jpg
blur img1 img1b
s img1b image2.jpg
q
```
